Minimalistic news agregator

## Installation

### Linux
1. Clone the source locally:

```sh
$ git clone https://github.com/montecamo/newsify
$ cd newsify 
```

2. Install project dependencies:

```sh
$ npm install
```

3. Start the site:

```sh
$ npm run serve 
```
